POST /v1/chat/completions

Generate AI chat responses using advanced language models. This endpoint provides access to 11+ premium AI models including Grok-3, GPT-4, LLaMA, Qwen, and DeepSeek for intelligent conversations, text generation, and question answering.

Request Parameters

Parameter Type Required Description
model string Required The AI model to use for completion
prompt string Required The input text to generate a response for
system_prompt string Optional System prompt to set AI behavior (default: "You are a helpful AI assistant.")

Available Models

ExoML Models

qwen-3-235b llama-4-maverick llama-4-scout gpt-4.1 deepseek-r1 gpt-4o gpt-4o-mini qwq-32b

Grok Models

grok-3 grok-3-fast grok-3-mini

Total: 11 chat models available. Models are automatically routed to the appropriate provider (ExoML or Grok) based on selection.

Examples

cURL
bash 
curl -X POST "https://user669-omniapi.hf.space/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "grok-3",
    "prompt": "Explain quantum computing in simple terms",
    "system_prompt": "You are a helpful physics teacher who explains complex topics simply."
  }'
Python
python 
import requests

url = "https://user669-omniapi.hf.space/v1/chat/completions"
data = {
    "model": "grok-3",
    "prompt": "Explain quantum computing in simple terms",
    "system_prompt": "You are a helpful physics teacher who explains complex topics simply."
}

response = requests.post(url, json=data)
result = response.json()
print(result["response"])

POST /generate

Generate high-quality images from text descriptions using advanced AI models. Supports multiple image generation models with various sizes and styles.

Request Parameters

Parameter Type Required Description
prompt string Required Text description of the image to generate
model string Required Image generation model to use
size string Optional Image size (default: "1024x1024")

Available Models

flux.1-dev flux.1-schnell flux.1-pro exo-image gpt-image

Examples

cURL
bash 
curl -X POST "https://user669-omniapi.hf.space/generate" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A futuristic cityscape at sunset with flying cars",
    "model": "flux.1-dev",
    "size": "1024x1024"
  }'

python 
import requests

url = "https://user669-omniapi.hf.space/generate"

POST /websearch

Search the web using OmniAPI uses Google Search for reliability. Can also scrape content from specific URLs using smart url detection.

Request Parameters

Parameter Type Required Description
query string Required Search query or URL to scrape
max_results integer Optional Maximum number of results (default: 10)
region string Optional Search region (default: "us")
safesearch string Optional Safe search setting (default: "moderate")
max_chars integer Optional Max characters for URL scraping (default: 2000)

Examples

Web Search
bash 
curl -X POST "https://user669-omniapi.hf.space/websearch" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "artificial intelligence trends 2024",
    "max_results": 5,
    "region": "us"
  }'
URL Scraping
bash 
curl -X POST "https://user669-omniapi.hf.space/websearch" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "https://example.com/article",
    "max_chars": 3000
  }'

POST /ttsgenerate

POST /transcribe

Transcribe audio files to text using AI transcription service. Supports various audio formats including MP3, WAV, M4A, AAC, OGG, and FLAC with high accuracy speech-to-text conversion.

Request Parameters

Parameter Type Required Description
audio_file file Required Audio file to transcribe (max 50MB)

Supported Audio Formats

MP3 WAV M4A AAC OGG FLAC

Examples

cURL
bash 
curl -X POST "https://user669-omniapi.hf.space/transcribe" \
  -F "audio_file=@/path/to/your/audio.mp3"
Python
python 
import requests

url = "https://user669-omniapi.hf.space/transcribe"

with open("audio.mp3", "rb") as audio_file:
    files = {"audio_file": audio_file}
    response = requests.post(url, files=files)

result = response.json()
if result["success"]:
    print("Transcription:", result["transcription"])
else:
    print("Error:", result["message"])
JavaScript (Browser)
javascript 
const fileInput = document.getElementById('audioFile');
const file = fileInput.files[0];

const formData = new FormData();
formData.append('audio_file', file);

const response = await fetch('https://user669-omniapi.hf.space/transcribe', {
  method: 'POST',
  body: formData
});

const data = await response.json();
if (data.success) {
  console.log('Transcription:', data.transcription);
} else {
  console.error('Error:', data.message);
}
